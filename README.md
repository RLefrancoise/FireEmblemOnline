# FireEmblemOnline

This is an old project. It is a browser online game inspired by Fire Emblem: Radiant Dawn. It uses CodeIgniter as framework.

There are many many things that still need to be done.

## What is done

- Database
    - Account
    - Unit
    - Items
    - Weapons
    - Class
    - Affinities
    - Biorhythm
    - Factions
    - Race
    - Skills (partially)
    - Trait
    - Unit bonds
- Pages
    - Unit status
    - Inventory
    - Shop

## What is missing

- Forge system
- Battle system
- Skills management
- Experience management (same as original game)
- Support management (same as original game)
- Home page content
- Account management
- About page content
- Forum
